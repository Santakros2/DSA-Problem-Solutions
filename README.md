# DSA-Problem-Solutions

Welcome to my **Data Structures & Algorithms (DSA) repository**!  
This repository contains solutions to various coding problems implemented in **Java**. It is organized by topic to help recruiters and fellow developers navigate easily.

---

## **Topics Covered**

- **Arrays** – Two Sum, Maximum Subarray, Sliding Window problems, etc.  
- **Strings** – Longest Palindrome, Anagram Check, String Compression, etc.  
- **Linked Lists** – Reverse Linked List, Detect Cycle, Merge Two Lists, etc.  
- **Stacks** – Min Stack, Next Greater Element, etc.  
- **Queues** – Sliding Window Maximum, Circular Queue, etc.  
- **Trees** – Binary Tree Traversal, Lowest Common Ancestor, Diameter of Tree, etc.  
- **Graphs** – BFS, DFS, Shortest Path (Dijkstra), Detect Cycle, etc.  
- **Dynamic Programming** – Coin Change, Longest Common Subsequence, Knapsack, etc.  
- **Backtracking** – N-Queens, Sudoku Solver, Subsets/Permutations  
- **Searching & Sorting** – Binary Search, Merge Sort, Quick Sort, Heap Sort  
- **Others** – Bit Manipulation, Union-Find, Sliding Window Techniques
